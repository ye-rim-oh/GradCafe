# GradCafe 2020-2026 정치학 박사과정 트렌드 분석

[English](README.md)

이 프로젝트는 2020년부터 2026년까지 GradCafe에 익명으로 보고된 미국 및 세계 주요 대학 정치학/국제학 박사과정(PhD) 입학 결과를 수집하고 분석하는 대시보드입니다.

## 빠른 시작

### 1. 전제 조건 (R 및 패키지 설치)

```r
install.packages(c("rvest", "httr", "dplyr", "tidyr", "lubridate", "stringr",
                   "plotly", "ggplot2", "rmarkdown", "knitr", "kableExtra",
                   "shiny", "shinyjs", "shinyWidgets", "DT"))
```

### 2. 바로 실행하기 (크롤링된 데이터가 있는 경우)

`scraped_2020_2026_combined.Rdata` 파일이 이미 준비되어 있다면 아래 한 줄로 대시보드를 바로 실행할 수 있습니다:

```r
Rscript -e "shiny::runApp('app.R')"
```

### 3. 전체 데이터 새로 크롤링하기 (업데이트 시)

언제든지 스크립트를 재실행하여 GradCafe의 가장 최신 실시간 데이터를 다시 받아 분석할 수 있습니다. Shiny 대시보드는 새로 구워진 `.Rdata` 파일을 즉시 반영합니다.

```r
Rscript scrape_all_years.R          # 전체 기간(2020-2026) 크롤링 수행 (약 5~15분 소요)
Rscript -e "shiny::runApp('app.R')" # 대시보드 실행
```

## 주요 파일 설명

| 파일                                 | 설명                                                                                                                                                                                       |
| ------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |

| `[sample] PhD Admission Analysis.md` | 수집된 데이터를 바탕으로 작성된 연간 동향 분석 보고서 영문 샘플.                                                                                                                           |
| `README.md`                          | 이 이정표(영문 텍스트).                                                                                                                                                                    |

## 대시보드 구조 (`app.R`)

- **Timeline (타임라인 탭)**: 1월부터 4월까지 대학별 합격, 불합격, 인터뷰, 대기(Waitlist) 결정이 언제 났는지를 Dot-plot 타임라인으로 보여줍니다. (마우스 호버 시 GRE/GPA/세부 분야 표시)
- **Trends (연도별 및 국적별 트렌드)**: 각 대학의 연도별 합격률 변화 추이와 미국(American) vs 유학생(International) 간의 합격률 격차 비교.
- **Subfields (세부 전공 분야 분석)**: 분야별(비교정치, 국제관계, 미국정치, 정치이론, 방법론 등) 지원자 수의 변화 및 각 세부 전공의 합격률 그래프 창.
- **Data (전체 데이터 테이블)**: 조건별 검색 및 열 정렬이 가능한 전체 데이터 표 기능. 행을 선택하면 구체적인 합불 노트(Note 코멘트 등)를 확인할 수 있습니다.

## 한계 및 주의사항

- **익명 자기 보고 데이터**: 본 프로젝트의 모든 데이터는 GradCafe 익명 사용자의 자발적 보고에 전적으로 의존합니다. 따라서 표본 편향(Sampling bias)과 정보의 과장, 누락 등 독립적으로 확인 불가한 오류가 포함되어 있을 수 있습니다.
- **크롤링 및 보정 오류의 가능성**: 학교 이름을 최대한 깔끔한 대표 형태로 묶고 세부 전공을 분류하기 위해 여러 R 정규표현식 루틴이 개입하나, 파싱 불완전성 혹은 예상치 못한 예외 항목이 소수 존재할 수 있습니다. 공식 통계가 아닌 '**대략적인 분석 참고용 지표**'로 활용하시기 바랍니다.
- **Accept Rate 계산식**: 실질적인 합격률 파악을 위해 `(합격) / (합격 + 불합격)` 공식을 적용하였습니다. (인터뷰나 Waitlist 상태인 경우 계산 분모에서 일시적으로 배제됩니다.)
